Overview

The Minesweeper game is a classic puzzle game that originated in the 1960s. It was popularized by Microsoft when it was included in the Windows operating system starting from Windows 3.1.

The objective of the game is to clear a rectangular board containing hidden "mines" or bombs without detonating any of them. The board is divided into cells, and each cell may contain either a mine or a number indicating the number of mines in adjacent cells.

The game is won when all non-mine cells are revealed. However, if a player clicks on a cell containing a mine, the game is lost.

Enjoy playing Minesweeper!

Instructions

To play Minesweeper, follow these instructions:

  1. Click on a cell to reveal its content.
  2. If a cell contains a number, it indicates the number of mines in adjacent cells.
  3. If a cell contains a mine, the game is over.
  4. To flag a cell suspected to contain a mine, right-click (or long press on mobile).
  5. If all non-mine cells are revealed without detonating any mines, you win!

Good luck!
